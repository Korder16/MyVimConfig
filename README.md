## Установка конфигурации
Для установки конфига достаточно запустить один из скриптов установки:
* Bash-script
```console
foo@bar:~$ ./install_config.sh
```
* Python-script
```console
foo@bar:~$ python3 install_config.py
```

## Установка плагинов
Необходимо прописать команду внутри редактора:
```console
:PlugInstall
```
